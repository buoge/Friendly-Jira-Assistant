## feat-001：增加任务拆分模板
比如： web新增组件的模板，通用模板，使用 spec 模式模板 等等，个人可以根据任务的需要按模板的逻辑快速拆分任务。


### 功能-1：新增模板
1. 在全局菜单左侧 TOOLS 下 新增一个菜单项“任务拆分模板”
2. 点击菜单项后，进入模板管理页面
3. 在模板管理页面，可以查看已有的模板，并可以点击模板名称进入模板详情页面
4. 在模板详情页面，可以查看模板的详细信息，包括模板名称，模板描述，子任务拆分样例等，后续进入后可以编辑更新

### tab 详情展示和创建模板
1. 列表详情以 tab 方式展示，多个详情可以切换
2. 在 任务拆分模板这几个字的右侧增加一个蓝色背景的按钮：新增拆分模板，用于创建新模版

### 详情页操作
* 在模板预览的右侧增加编辑按钮，点击编辑后在右下方增加编辑按钮。
* 模板通过拖动可以排序，拖动过程中在 2 个按钮之间来一个位置指示线。
* 取消按钮放在 update 按钮的右侧，用于取消编辑。
* 在编辑按钮右侧增加一个红色的删除按钮用于删除模板，删除时需要二次弹窗确认
* 通用模板 这个不能删除

### 功能-3：应用任务拆分模板
* 在"批量创建子任务" 的右侧增加一个按钮："应用任务拆分模板"按钮
* 点击 "应用任务拆分模板" 按钮后，弹出一个模态框，模态框中显示模板内容，并可以点击“使用模板”按钮。
* 点击“使用模板”按钮后，模态框中显示模板内容，内容要求以表格的形式展示每一项内容，如 任务名称：xxxx 任务；初始预估：0小时
* 模板预览弹窗下方可以点击“生成任务”按钮，把内容写填到批量内容创建的输入框内
* 初始预估小时这里要做校验需要大于 0 才可以，点击生成任务的时候校验
* 生成任务时已经有的子任务不在重复创建

### 功能-4：应用模板预览弹窗内的子任务调整
应用模板预览弹窗内
* 新增：在当前行下方插入一条空子任务（任务名称为空，初始预估为 0），并自动聚焦到新行的名称输入框
* 删除：移除当前行；仅剩一行时删除按钮禁用，保证至少保留一项
* 现在展开的故事默认时编辑状态，现在期望在 当前状态 “打开的” 按钮右侧 增加一个按钮 “预览模式”，打开此模式后
